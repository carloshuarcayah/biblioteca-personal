# Biblioteca Personal API

## Descripción
Esta es una API RESTful diseñada para gestionar una biblioteca personal. El objetivo principal de este proyecto es aplicar y demostrar fundamentos sólidos de desarrollo Backend, incluyendo arquitectura de capas, manejo global de excepciones, operaciones con bases de datos y despliegue en la nube.

## Stack
* **Lenguaje:** Java 25 LTS
* **Framework:** Spring Boot 4
* **Base de Datos:** MySQL 8.4
